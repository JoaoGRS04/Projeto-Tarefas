# Gerenciador de Tarefas
O projeto é um gerenciador de tarefas simples feito em Python utilizando interface de terminal (CLI)

Funcionalidades:
- Adicionar novas tarefas
- Armazenar as tarefas adiciondas
- Mostra a auta lista de tarefas
- Removar uma tarefa

## Instruções
Ao executar o programa, será exibido um menu com as seguintes opções: 
1. Adicionar nova tarefa 
2. Listar tarefas 
3. Remover uma tarefa 
4. Sair

Logo abaixo do menu, o usuário deverá digitar o número correspondente à opção desejada.
